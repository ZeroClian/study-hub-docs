## 计算机组成与体系结构

1. DMA控制方式是在**主存与外设**之间**直接**建立数据通道进行数据的交换处理。
2. 在微机系统中，BIOS保存在主板上的**ROM**中。
3. CPU在执行指令的过程中，会自动修改程序计数器（PC）的值，以使其保存的总是将要执行的下一条指令的地址。
4. 程序计数器（PC）用于存放下一条指令所在的单元的地址。当执行一条指令时，处理器首先需要从PC中取出指令在内存中的地址，通过地址总线寻址获取。


## 数据的表示

1. 正数的原码、反码、移码相同
2. 负数的反码=原码除符号位外按位取反
3. 负数的移码=反码+1
4. 1 B = 8 bit；1 KB = 1024 B
5. 最小存储单位为B（字节Byte），最小数据单位为 bit（位）
6. D：10进制；B：2进制；O：8进制；H：16进制


## 端口

常用协议端口号情况如下：
- POP3:TCP的110端口，邮件收取。
- SMTP:TCP的25端口，邮件发送。
- FTP:TCP的20数据端口/21控制端口，文件传输协议。
- HTTP:TCP的80端口，超文本传输协议，网页传输。
- DHCP:UDP的67端口，P地址自动分配.
- SNMP:UDP的161端口，简单网络管理协议。
- DNS:UDP的53端口，域名解析协议，记录域名与IP的映射关系。
