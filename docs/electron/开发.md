使用 fortawesome 

- 引入

```
import { faSearch, faTimes } from '@fortawesome/free-solid-svg-icons'
```

- 使用

```javascript
<FontAwesomeIcon 
    title='搜索'
    size='lg'
	icon={faSearch} />
```

使用PropTypes进行类型检查

- 引入

```javascript
import PropTypes from 'prop-types'
```

- 使用

```javascript
FileSearch.prototypes = {
  title: PropTypes.string,
  onFileSearch: PropTypes.func.isRequired
}
```
